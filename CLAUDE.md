# Claude Code Instructions

## Git Commit Guidelines

When creating git commits:

- **DO NOT** add "Co-Authored-By" lines to commit messages
- Keep commit messages clean and concise
- Follow conventional commit format when appropriate (e.g., `feat:`, `fix:`, `chore:`)

## Project Context

This is a Chrome extension for organizing browser tabs. The extension:

- Sorts and organizes tabs
- Manages tab groups
- Stores user preferences using Chrome's storage API
- All operations are local (no external servers)
